## nodeJs+express4写一个小后台

一个简单带有登陆功能、增删改列表功能的小后台（虽为前端，这个项目没做样式美化，基础bootstrap样式引用），全面应用nodeJs+express4框架技术点

### 项目运行：

1、安装依赖：npm install

2、运行：npm start

3、项目运行地址：http://127.0.0.1:3000/
