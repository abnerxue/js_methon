
##### VeeValidate在vue里的表单验证示例
VeeValidate是Vue.js的验证库，它有很多验证规则，并支持自定义规则。它基于模板，因此它与HTML5验证API相似并且很熟悉。您可以验证HTML5输入以及自定义Vue组件。它也是以本地化为基础构建的，有大约44种语言由精彩的社区成员支持和维护

######包含以下几点应用：
* 1、基本安装使用
* 2、本地化支持
* 3、自定义规则和错误消息
* 4、验证HTML5输入和自定义Vue组件
* 5、统一提交按钮处理
